MÓDULO PADRÃO WORDPRESS LESS-IS-MORE 1.0

Instalação.

Para criar uma instalação nova do wordpress leia o tutorial em: http://br.wordpress.org/.

Após criado o site em wordpress(wp), crie uma pasta dentro de /wp-content/themes com o nome do seu tema.
	Copie os arquivos contidos nesse repositório na pasta criada
	Edite o arquivo /inc/config.php com o nome do tema, track do google analitycs e os css/javascripts que o tema irá usar.
	
Após editar o tema
	Crie um print do site e salve na raiz do tema com o nome screenshot.png
	Edite o style.css com as informações do tema.
